tagattr(name, expr, value=nil, escape=true)

(experimental) return ’ name="value"’ if expr is not false nor nil. if value is nil or false then expr is used as value.

[Source]

# File lib/tenjin.rb, line 62
    def tagattr(name, expr, value=nil, escape=true)
      if !expr
        return ''
      elsif escape
        return " #{name}=\"#{escape_xml((value || expr).to_s)}\""
      else
        return " #{name}=\"#{value || expr}\""
      end
    end
