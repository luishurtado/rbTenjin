Class Tenjin::ErubisTemplate
In: lib/tenjin.rb
Parent: Tenjin::Template

template class to use eRuby template file (*.rhtml) instead of Tenjin template file (*.rbhtml). requires ‘erubis’ (www.kuwata-lab.com/erubis).

ex. 

  require 'erubis'
  require 'tenjin'
  engine = Tenjin::Engine.new(:templateclass=>Tenjin::ErubisTemplate)
